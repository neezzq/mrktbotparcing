Это версия без платного Gift Asset API.

Что работает:
- MRKT через auth + gifts/saling
- /start
- inline-меню фильтров
- фильтр по подарку / модели / фону / цене
- фильтр по маркетам через кнопки
- рассылка всем подписчикам

Важно:
- Реально реализован только MRKT как источник листингов.
- Кнопки Tonnel / GetGems / Portals / xGift уже есть в меню и сохраняются в фильтрах,
  но для полноценного подключения этих источников нужны их рабочие публичные endpoints или парсинг.
- Avg sell здесь считается по похожим активным листингам MRKT.
- Avg buy пока не доступен без отдельной истории сделок.
